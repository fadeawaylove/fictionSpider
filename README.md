## 简介

免费的小说采集程序，只需输入小说网址，则自动抓取小说内容，并生成epub文件。

## 使用

```shell
# 安装依赖
pip install -r requirements.txt
# 运行
python main.py
# 然后根据提示输入小说地址
```
